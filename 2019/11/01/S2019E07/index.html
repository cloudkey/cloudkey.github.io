自学类专业网站

2019.11.01 Jun Ni read  热度 ℃

[TOC]

之前看到一个关于自学的观点，跟着兴趣走，定下一个方向去规划学习。可能不知不觉你就成为了某个领域的专家。

转自互联网上的一份很全的自学类专业网站。
都是硬干货。有资料类的，还有视频类的，有趣且有用。

你可以学习一切你所热爱的，只要你想。

历史文学类

  1. 中国妖怪百集
    “妖怪控”可以看这里。
    一个收录了中国古今妖怪的网站，有所有妖怪的信息、出处。

  2. 全历史
    非常完整的世界史百科全书～历史，古籍，古迹，甚至还有宇宙史。
    比较厉害的功能是AB路径，比如你输入两个完全不相干的人名，它就会给你生成一张关系图谱，有些八竿子打不着的人都能找到联系。
    还有一个全画作功能，就是你输入一个人名，就能找到他的全部作品。

  3. 发现中国
    各式各样的地图，有趣的点在于主题非常丰富，比如：
    金庸宇宙地图，水浒地图，冰火世界地图。还有珠峰攀登者路线，小说里出现的奇幻动物分布地图，甚至还有上海的旧马路地图。

  4. 书格
    专门提供古籍阅读的网站。

影视频

  1. 纪录片
    里面有3000部左右纪录片，国内外都有，很多别的网站上找不到的可以来这里找。全部免费观看——视频可以直接保存到网盘。

  2. Crash Course 中文字幕组
    几百集国外高质量视频课程，从社会学，文学到工程学，计算机，表演……都是中文字幕。
    网站里的视频课程也一直在更新。

  3. 译学馆
    专门翻译搬运国外优秀视频，演讲。
    这个网站里，你也可以申请成为译者，帮忙翻译字幕，也算是英文练习了。

  4. Oeasy
    完全免费的专业视频技能学习网站，界面简单到土，但是教程丰富。
    包括PS 教程、手机摄影教程、Ai 做图教程、Excel 教程、Word 教程、PPT 教程、Pr 视频剪辑教程、Ae 视频特效教程、Au 音频教程、Flash 教程、做网页……

  5. 爱琳的挑战
    视频自学简单的日语，有各种短剧，都是比较生活和日常的对话内容，实用性还蛮高的。

  6. 钢琴
    想学钢琴的可以看看这个网站。
    里面有许多钢琴视频课程，包括入门、考级等，以及琴谱。
    也有许多表演视频，如果不想学欣赏也不错。

  7. 自由钢琴
    练习的话可以用这个。可以用电脑键盘模拟弹钢琴哈哈哈哈哈。

  8. 免费吉他课
    免费的基础吉他相关知识，起步课。可以自己买了吉他抱着练～

医学、法学、金融、建筑学

  1. 默沙东诊疗手册
    医学自学网站。这是世界使用最广泛的医学信息资源之一，分为专业版、大众版、兽医版。
    专业版里还有各种医学操作方法和视频，照片、案例、数据、图像、新闻等等……
    但对于一般人来说，大众版就够了，多学一些权威的医学常识，避免百度得到混乱信息。

  2. 医学微视
    分享各种医学知识的网站，由中国医学科学院健康科普研究中心监制，不卖产品。
    里面会用短视频的形式帮助你了解各种疾病和医学知识。

  3. 法律图书馆
    如果想学习一些法律常识或者遇到了一些法律问题想要搜索的，可以用这个。
    里面还有大量的论文，按照不同法分类。

  4. Goodbuildings
    世界建筑作品大全。收录了非常全的建筑物资料和案例，很多都是大师级作品。对这方面感兴趣的可以多看看。

  5. FT中文网
    学习金融知识。里面是来自英国《金融时报》权威性全球财经新闻，拥有关于金融最权威的分析以及评论。

科普、工具

  1. Sciencedaily
    美国科普文章专题新闻网站，覆盖面很广。
    包括：天文学、太阳系外行星、心理学、社会学、人类学、生物、地质、气候、空间、计算机科学等等各种各样的科学课题文章。

  2. 盒子学术
    这个怎么说呢，就是「万能滑滑梯」。
    有音乐，图片，电影，论文……反正你想要找的一切都找得找。

  3. 反向搜图
    也是工具类。比较小众的搜图网站？我觉得效果很不错。还有一个“灵感搜索”功能，就是你如果找不到这张图了，也可以输入你脑里模糊印象的关键词，他就会帮你找到相关的图片。

  4. 彩云小译
    可以翻译整篇文档，直接将文档上传到网站，就能翻译了。翻译结果还会直接嵌在英文下方。
    还能直接上传视频翻译字幕……

  5. Openstax
    免费的国外教科书。可以用网页的翻译功能翻成中文，也不用注册，可以直接下载，还有目录。
