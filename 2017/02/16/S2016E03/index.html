喜出望外中签WMM东京马拉松

2017.02.16 Jun Ni run  热度 ℃

[TOC]

再过两天我就要启程去日本参加2017年的东京马拉松了，虽从年前就开始准备相关事宜，但是临近出发还是抑制不住内心的小激动啊。
这一篇回顾我的2016不光说一说中签东马，还想聊一聊这些年跑过的各个城市的马和发生的路上的那些趣事和见闻。

相信每一位跑马的小伙伴都知道WMM，即World Marathon Majors（马拉松6大大满贯赛事），是世界上最顶级的六大马拉松赛事了，每一家都有各自的特色，所以对于每一位认真跑马的同学来说，能有资格参加，无疑是人生最享受的旅程之一了。

世界六大马拉松

之前的一篇博客介绍过4大慢跑鞋，这里也简单介绍一下WMM吧。
世界六大大满贯分布在4个国家，分别是：

美国：纽约马拉松，波士顿马拉松，芝加哥马拉松
英国：伦敦马拉松
德国：柏林马拉松
日本：东京马拉松

纽约马拉松 - 世界最大

纽约马拉松于1970年由纽约路跑者组织发起，每年都选11月初刚进入秋天的时候举办，平均每年约有3万人左右报名参加，名列世界六大马拉松赛事之一。纽约马拉松从史坦登外岛出发，沿途经过纽约市的五大区，一路上可以感受各区不同的都市面貌与文化，最后终点则跑入纽约著名的中央公园，跑完一场纽约马拉松也认识了纽约的「一个大苹果」文化。2013年创下高达5万人以上完成全程马拉松的壮举，为世界上最大一场的马拉松纪录。

波士顿马拉松 - 历史最久

波士顿马拉松名列世界六大马拉松赛事之一，主办单位为波士顿体育协会创办于1897年，是历史上最悠久的年度马拉松。每年皆在四月的第三个星期一，美国爱国者节那天举行。跑者们由麻州西部的霍普金顿出发，沿著公路往波士顿市中心前进，终点就在波士顿公立图书馆旁边的科普利广场，路线起伏颇大。该赛有一项最为人知的事便是它严格的速度门槛（被业界称为BQ），要报名参赛的跑者必须达到门槛才能参赛。即便非常困难，仍让不少人趋之若鹜，每年平均仍有两万多人报名。波马一般是所有跑马小伙伴的最终极目标了，被称为跑马者心中的麦加也不为过。
目前百度内部跑步协会有一位16年去参加了波马（当然不是慈善名额哈），已被大家封神，可见波马在大家心中的地位。

芝加哥馬拉松 - 全美管理最好

芝加哥马拉松，与纽约马、最古老的波士顿马同在美国，并列世界六大马拉松。对于波士顿马拉松有速度门槛的限制，芝加哥马拉松就相对亲切许多，除了没有速度门槛之外，芝加哥马同时也是IAAF金牌赛事，此外，年年参赛者当中更有45%的跑者为女性，还有强大的配速团队指引想要有所突破的跑友。不难发现无论赛道延伸到哪儿，总有来自各个国家的热情民众，声嘶力竭地为跑者鼓舞，在这场国际级的赛事中，看不见人车争道的惊险镜头，也没有人潮拥塞的赛道。
今年我所在的团队有两位小伙伴中签了17年的加哥马，预祝他们取得好成绩！

伦敦马拉松 - 世界最大慈善募款

伦敦马拉松与波士顿马拉松一样，都在气候宜人的四月份举行，最初创办的缘由，是两位英国人参加纽约马拉松后，有感于城市马拉松的欢乐团结气氛，因此决定创造属于伦敦的马拉松庆典。自1981年开办以来，受欢迎的程度不输纽约马，年年报名人数相当可观。伦敦马拉松的特别之处，就是参赛者除了可以报名抽签取得参赛资格，还可透过募款方式，相当有趣。这项结合公益，让伦敦马拉松成为今天世界纪录上最大的募款活动。

柏林馬拉松 ─ 世界最快

号称史上最速赛道的柏林马拉松由于赛道相对笔直、平坦，许多长跑健将都在柏林马拉松中改写世界纪录，因而被世人称作「最快的马拉松路线」，名列世界六大马拉松赛事之一，每年都在9月的最后一周举行。起点与终点设置在勃兰登堡门，绕行市区主要干道，穿越柏林东西两边知名的景点，包含历史古迹和东西德统一后新建的现代化建筑，极具有历史意义。

东京马拉松 - 亚洲最大

东京马拉松从2007年开始将加入世界马拉松系列赛的行列，成为波士顿、伦敦、芝加哥、纽约、柏林之外的第六名成员。每年将会有超过35,000名跑者从东京都厅出发，赛道绕行各观光景点，跑者奔驰在东京的街道上亦可体会到不同的城市路跑风情。赛道沿途更透过音乐、舞蹈、民俗技艺等多元表演活动，集结众人之力共襄盛举，让东京的街道成为跑者与加油民众皆能投入参与的舞台。


而当你完成这六大大满贯所有赛事的时候，你将会收到一份特殊的礼物，铛铛铛 — 那就是六星奖牌，以奖励你对马拉松的热爱和坚持。

从报名到中签 - 东马

大多数中国跑马的小伙们的第一个WMM都会选择东马，一方面是东京离中国较近，另一方面东马赛事的口碑也是非常好的，每年报名的人数在30W+，而中签的大陆选手在800-1000左右，所以中签率是非常低的，但这并不影响大家对东马的向往和热情。

每年的东马在8月底左右开始报名，今年也不例外，和组里的跑马小伙伴商量了一下决定试一试，如果中签了那就去见识见识。
在填写完注册信息后，会收到下面的邮件，表示注册成功，等待抽签即可，报名费折合人民币大概800元。

大概过了一周左右，就收到了东马组委会收到的本届报名的统计情况，比如17年东马的报名总人数是：322,703，其中全程 321,459，10公里 1,244。

又再过了两周左右，就收到了一封Congratulations邮件，哈哈哈，看到了这个单词就表示中签啦，但是如果未中签，那就是Thank you letter了。

但是同组其他报名的小伙伴都未中签，哎，可惜没办法一起约跑东马了。。就这样，历经了3周左右的时间从报名到中签，完成了去往东马的第一步，接下来就是各种准备攻略了，芽庄之旅后，深知又是一项工程啊。。。
