全日空航空

熟悉的陌生人

提起日本，一方面能想到 村上春树、川端康成、大江健三郎、黑泽明、宫崎骏、久石让等，另一方面又会联想到南京大屠杀、钓鱼岛、APA HOTEL事件…

中日隔海相望，一衣带水，但是对于咱们这个邻居，却用着一种非常复杂难以言表的感觉：距离上很近但是心里上感觉又很远，亲疏上既熟悉又陌生。

Outline

签证

对于日本签证而言，因为不需要面签，所以个人没有送签权，要拜托使馆指定的旅行社代办机构办理。我是在淘宝选的一家评价较高的店办理的，有单次个人旅游签（最长停留15天），三年多次签，5年多次签。在北京大使馆办理日本签证大致需要以下这些材料，大概两周出签：


信用卡


又要跟大家安利这张16年的神卡了，这次出国薅了不少羊毛。

  1. 航班接送机：北京走的神舟租车，服务很赞，准时接送。
  2. T3航站楼的贵宾厅早餐不错，可通过Priority Pass APP查询世界各地机场可使用的贵宾厅情况，有的比较适合商务，有的适合休息，而有的餐食很丰富。
  3. 基本在日本所有刷卡都走的AE通道，境外1%的返现，另外3.31之前境外消费1W以上，超过1W部分返现10%，最高返现1000元人民币。
  4. 浦发的这张卡非常遗憾缺少安检快速通道功能，从北京出境的人每天还是非常多的，建议大家在携程上购买快速通道权益，可以减少不少排队等待的时间。

Day 1

此趟我们的航班是选择的全日空航空的NH964／963班次（波音787梦想航空），每年全日空会在双11附近做活动，中国多个城市飞日本多地的特价机票，去年飞猪和全日空合作还可以用淘宝的双11券，我买的北京来往东京大概只花了1200左右（含税），所以应该说是白菜价了。
这里去日本推荐一下全日空航空(首架波音787于2011年9月26日交付全日空航空公司使用)，全日空是“星空联盟”航空联盟成员之一。截至2007年3月，全日空共有22,170名雇员。1999年10月，全日空正式加入星空联盟，全球9家5星航司之一：
Skytrax五星航司





早晨8:30（北京时间）从首都机场T3起飞，下午12:45（东京时间）抵达东京羽田机场，之后过海关进行安检。
但是在等行李的大厅发生了一件小意外，早晨出发的时候我在书包了塞了一个苹果准备在飞机上吃，但是忘了吃，在等行李的时候机场的警犬老远就闻到了我书包里携带有违禁物品，从远处一路奔过来，在我们身边一直转悠，最后机场安检人员让我们到附近的检查处检查，将我们苹果收走了，估计是防止物种入侵吧。


过关后，2020东京奥运的气息就扑面而来，突然让人联想到了2008北京奥运，期待2020.

在东京羽田机场逛了两小时，就准备搭乘ANA日本国内航班前往大阪了，大概1个半小时抵达大阪关西机场，之后乘坐南海小火车去往大阪市中心的酒店。

在日本购买地铁票，是先看售票机器上的地图，地图会显示从当前地点到目的地所需的票价，然后在机器上选择购买多少钱的票价，投入钱币，就会出票了，跟国内在机器上选择目的地不一样。





晚上入住酒店，酒店房间真的小到一点空间都不浪费的地步了，但是设施尽然都很全，小冰箱+咖啡机+电动马桶，房间跟之前在网上查的一样很干净。
