第一次出国越南芽庄的难忘之旅

2017.03.28 Jun Ni travel  热度 ℃

[TOC]

今天终于要开始写芽庄之旅了，时间真快，转眼已经过去快小半年了，但是这次出行给我留下了非常多难忘的经历，也对以后我的旅行影响深刻，所以容我慢慢回忆，当然有很多非常好玩和有趣故事，也会首次讲述一些大伙不知道的“惊险”时刻。

这篇芽庄之行就不以游记的形式来写了，东京游记篇还是热乎乎的可以去那里逛逛，这里我想以一名团队出游组织者的视角来讲述此趟出国的所见所闻所感。

出发前的准备

距离上一次 team building 差不多过去快两年了，期间组里也有讨论过出行的计划，但由于种种原因未能成行。大概去年暑假的时候，小伙伴们渐渐又开始讨论这个话题了，可能是两年都没出去了，所以这一次大家觉得是时候玩一次大的了。
时间来到八月，躁动的心又开始蠢蠢欲动，在一次中午出去吃饭的路上，我一时兴奋撒出豪言“这次出行，我来搞！”，其实当时脑子里还是一片空白没有任何经验和想法，不管了试试看吧，就这样我 start run!

为啥开始要出国了

还记得那会网上流传着思聪爸豪的那个“先挣个它一个亿的小目标”视频，所以当时我的第一个想法是咱们能不能也定一个“出国bui的小目标呢”。

其实细细想来也未不可行，首先周边团队已有出国bui的先行者了，据说上线效果不错；其次PCS之前几年每年虽都有出行，但还未踏出国门，所以做一次全新的尝试应该很有意思；最后从实际考虑，其实现在国内想玩得好的话，花费也不小了，两年前的那趟三亚之行就已有感受，加之国内旅行的体验咱们都懂得，所以出国这个方案可行。

目的地的选取

虽然是出国，但也不是所有国家都可以考虑，当时记得大概有这么些个前提考虑因素：

  1. 尽量免签 或者 签证非常非常容易办的地方；
  2. 尽量去大家没有去过的地方，玩一些不一样的体验；
  3. 整趟旅行时间不能太久，毕竟年假有限；
  4. 由简入奢的过程（三亚经验），自选酒店不低于希尔顿的体验标准（怀念三亚的希尔顿啊！）；
  5. 经费人均5-6K，新入职的同学也可以一起high；
  6. 速战速决！

基于以上，综合调研结果，最终选定越南芽庄 - 东方马尔代夫。
并将此次出行的主题定为“一岛一酒店，极致奢华”。

在芽庄我们要玩什么

由于是团队整体出行，不同于个人旅行，所以在景点选取和玩的项目上需要综合考虑团队人员组成&&平常喜好&&氛围特性，当时我大概是这么想的：

  1. 首先无论是完全跟团游还是完全自由行都不太完美或者可行，跟团游可能玩得不尽兴 and 参与的可操作性较低；自由行吧，这多人出国安排着实不易（比如飞机，当地交通，餐食，门票，住宿，国内自己安排还好，海外并不容易）。所以准备采用半跟团半自由行的形式，希望可操作性较高的同时小伙伴们也能玩得爽。

  2. 在同程网上找了一个旅游顾问，他们给了一个已有的芽庄常用团游方案，在此基础上，根据“领略当地特色景点，海滨城市出海潜水必备，加入自由行行程去除重复吸引力较低的安排”，所以给出了 当地特色景点 + 出海游 + 自由行的最终方案。

  3. 其实在做方案前，我着重考虑的是自由行部分，因为跟团游无非就是景点项目这些，而自由行确实关乎着这趟旅行成败的关键。而在调研芽庄的时候，珍珠岛给我留下了很深的印象。

    它是俄罗斯的一个财团重金打造的度假之岛，离海边较远，风景海水沙滩皆属上乘，休闲度假很是惬意。

    岛上设施完备，高档酒店（海滨别墅，园景别墅，山景别墅应有尽有），游玩项目众多（游乐场，滑翔伞水上摩托游艇等各种水上项目，高尔夫水球沙滩排球等体育项目），还有一条世界上最长的海上缆车。

    岛上的服务一流赞，能够体验一把庄园式服务，从住宿餐食到游乐项目到岛上风景都引人入胜啊。传有，到芽庄不到珍珠不算来过。

  4. 我们team属于较为活泼的那种类型，并且之前有过三亚bui的经历，对团建的品质还是很有追求的，所以地点／景点／项目 选择上尽量“不落俗套，实际高端兼并，不忙不累休闲放松，体验异国风情”。

认识芽庄，走进东南亚


芽庄位于越南的东南海岸，庆和省省会，拥有全越南最好的海滩，是一座著名的海滨城市。这里地理位置优越，自然风光优美，并且早在越战时期就是美军的度假基地，所以虽属于共产主义国家，但芽庄却有着浓厚的西方风情，和越南的北越城市（河内，胡志明市）风格差别较大。


芽庄以渔业和旅游业为主要产业，和其他东南亚地区一样，整体消费水平较低（相对于天朝来说），城市整体基建大概相当于国内的三线城市水平。不过自然风景不错，没有雾霾，属于热带气候，所以冬季过来度假比较惬意。


当地老百姓的生活节奏感觉较慢，给人一种悠然自得的感觉，平常出行摩托车为主，不过大家都很守交通规则，开车基本都带安全帽。听导游说，这里的外来产品（汽车、家电）以日本、韩国较多，因为他们觉得中国的产品质量不是很好（所以，我们还需努力啊）。这里的治安整体感觉还是挺安全的，晚上出去喝酒吃饭，没有感觉很乱的样子。

芽庄宣传片，片中涵盖了此次我们的整个行程

踩了哪些坑

这趟旅行总体来说体验还是比较赞的，但是也踩了不少的坑。

非常清楚的记得10.20当天下午抵达天津机场候机的时候，收到一条重磅消息“360宣布推出个人云存储市场”，擦，第一感觉就是在外国估计我们也会有事干了，哈哈哈。

但这还不是最惊险的时刻，记得在柜台办理登机手续的时候，我听到有其他的团队说3G流量卡的事情，what？不是到了当地导游发放么？询问完领队之后，得知是出发前的一两周邮寄过来的，啊！！！当时真的有点懵了，这还没出发呢就已经出状况了… 我赶紧拨通之前旅行社顾问的电话，当时正直晚上吃饭时间，打了N个电话没人接，微信短信语言基本上全试了，还是找不到联系人…当时心想，阵脚先不能乱，电话继续打，团队办理登机手续继续办，就这样，大概过了二十分钟，旅游顾问的电话终于打了过来，一番沟通后，原来是旅行社忘了给我们邮寄，最终旅行社帮我们从其他领队那边紧急调拨了35张流量卡，赶在登机前顺利解决。

凌晨1点抵达芽庄金兰国际机场，过海关的时候，得知还需要给签证盖章工作人员小费（折合人民币10元左右），组里有同学试着不给但是被拒绝入境，让重新排队，第一次出国竟然遇到这事，算了，不跟他们一般计较了。

跟团的那几天吃饭都是地接安排的，整体餐食较差，口味分量环境都不是很好，跟国内旅行社反映后，只是稍有改善提升不大，以后团建出去玩吃饭问题需要重点考虑。

自由行是安排在珍珠岛上面，基本没啥问题，只是大伙拿着房卡去坐海上缆车返程的时候，发现刷卡显示的照片与本人不是对应的，被拒绝乘坐缆车返回。原来是我从前台拿卡的时候，是按照房间号拿的，并未与房间里的人对应，最后和工作人员沟通一番才得以过关。

最后一天返程前，按照之前的合约，需要随团去几个购物点，不过跟电视中看到的强迫购物新闻不同，此趟行程的几个购物点都安排了相关背景知识介绍和展览参观（越南沉香，珍珠，硅胶等），并没有出现强迫购物的情况，所以感觉没有那么差。

如何做一名合格的组织者


如果你问我作为一名团队的组织者，需要怎么做才能算合格？那么我会说：

耐心
足够的耐心
前面两件事很重要，重要的事再说三遍

从前期方案设计、反复推倒再重新规划，实施推进，过程中各种琐事，结束后的账单收尾，都需要组织者有足够的耐心，不能慌不能乱也不能烦，否则将会深陷泥涝，影响自己也会影响到同行小伙伴的心情，要有一颗“团结紧张活泼的心”（看似简单，实则不那么简单）！

战略上说完了，从战术上可以怎么做呢？我的经验是：

总体把握，局部放开，安全第一，尽high尽爽

首先顶层方案设计非常重要，关乎着整趟行程的成败与否，不能太低端也不能过于高端，需要兼顾普众化。

其次团队人员安全第一（尤其是海外，文化差异需要注意），随行的不仅包括你的同事可能还有他们的家人，老人孩子需要重点关注，这些在做方案设计的时候就需要考虑进去，否则后续战术上的勤奋很难弥补战略上的失误。

预算在出行中也是重要一环，需要组织者总体把握好，不能大手大脚心无胸竹，否则可能会出现超预算过多的尴尬局面，财务是组织者重要的一课。

若出行人员过多，小组分治法可以考虑，每组选一名小组长，负责点名／收发东西／活动住宿安排等，分而治之，效率效果都会比组织者单打独斗要好很多。

最后还有一点需放在心上，作为组织者要多考虑一些想得更远一点，任何事都要有头有尾，不能光顾自己玩忘了团队，每天早上出发前把今天的行程梳理一遍，晚上休息前将明天的行程安排再确认一遍，胸有成竹，自己玩得好大家也能玩得爽！

我的成长

不经意间，蓦然回首，慢慢发现：

自己的耐心更足了，性格也更耐磨了，做事考虑会多一点，“进程切换转为线程切换”得从别人的立场角度看待问题了。

以后外出旅行更加轻车熟路了（今年的日本自由行就有了经验），（机票，酒店，行程，景点项目，美食…）这些都已轻描淡写，享受旅行已成为主旋律。

组织能力得到了一次很大的锻炼（其实平常机会并不那么多的），说跟做还是差距很大的，尤其是涉及到人的事情，学会沟通，团结team，更好的融入大家，也提升了自己。

开拓视角、眼界，行万里路知万卷书，旅行不光是玩也能给你一个多元的世界，“江南无所有，聊赠一枝春”。

芽庄之行虽以过去半年，但是那些美好的画面快乐的时光仍然历历在目，而自己也在这趟旅程中收获良多，感谢team小伙伴们的支持也感谢自己点滴的积累！

华灯初上，祝大家晚安！
